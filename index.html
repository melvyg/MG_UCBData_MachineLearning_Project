Marvel Character K-means Clustering Using Pyspark and NLP

The goal of this project is to identify marvel characters with the largest comic influence or social status--use k-means clustering define and visualize different marvel universes and the connections between these universes. To achieve this, I use natural language processing on the comics, series, and event text provided by the developer Marvel API for each marvel character. Furthermore, I aim to create interactive D3.js visualizations to illustrate how k-means is implemented and how the marvel character clusters are defined. Also, for fun a force-directed bubble chart with bubbles scaled to the number of comics a character is a part to idenity the most popular marvel characters.
